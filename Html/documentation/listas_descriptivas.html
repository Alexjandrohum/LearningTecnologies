Lenguajes de programación
Java es un lenguaje de programación realmente tipado por sus explícitas estructuras se sintaxis.
Java Script es un lenguaje de programación no tipado ya que no se especifica el tipo de variables a utilizar.
Swift es un lenguaje de programación tipado y no tipado ya que se puede especificar o no especificar el tipo de variables sin tener un error de sontaxis.
Bases de datos
Son una estructura en la cual nos ayuda a almacenar información de un sistema para mantener de una forma más ordenada y optimizada.
